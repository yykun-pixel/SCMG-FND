# 视频分析提示词模板

本文档包含用于视频内容分析的提示词模板，用于虚假新闻短视频检测。

## 1. 默认分析提示词 (DEFAULT_PROMPT)

```
请对这个视频进行详细分析，重点关注虚假新闻短视频检测相关的特征，并严格按照下面指定的JSON格式返回分析结果。

你必须严格遵循以下JSON格式，不得修改键名或结构，不得添加额外字段：

{
  "虚假新闻短视频检测特征分析": {
    "信息真实性": "分析视频中信息的真实性，包括内容是否经过验证、是否存在误导性表述",
    "视觉内容分析": "分析视频画面是否存在人为编辑、合成、特效处理等痕迹",
    "叙事手法评估": "分析视频的叙事方式是否存在情感引导、偏向性表达等问题",
    "信源可靠性": "评估视频信息来源的可靠性，是否缺乏权威信源佐证"
  },
  "检测结论": "基于上述分析，对视频是否具有虚假新闻特征的综合判断"
}

重要提示：
1. 不要使用任何Markdown格式（如```json）
2. 不要在JSON前后添加任何额外文字
3. 你的回答必须是一个有效的、可直接解析的JSON对象
4. 不要更改JSON结构或键名
5. 每个分析字段必须包含实质性内容，不少于50字
6. 检测结论必须全面概括各方面特征，不少于100字

请确保分析客观、详细且有深度，不要遗漏任何可见的细节。返回的必须是格式正确的JSON，不包含任何额外的解释文本。
```

```

你必须严格遵循以下JSON格式，不得修改键名或结构，不得添加额外字段：

{
  "视频文字内容": [
    {"时间点": "开始时间-结束时间", "文字内容": "提取的文字内容1"},
    {"时间点": "开始时间-结束时间", "文字内容": "提取的文字内容2"},
    ...
  ],
  "总结": "对视频中所有文字内容的简要总结"
}

重要提示：
1. 不要使用任何Markdown格式（如```json）
2. 不要在JSON前后添加任何额外文字
3. 你的回答必须是一个有效的、可直接解析的JSON对象
4. 不要更改JSON结构或键名
5. 每个文字内容必须准确提取，不要添加不存在的文字
6. 总结部分应简要概括视频文字的主要内容，不少于30字

请确保提取的文字内容完整、准确，按照时间顺序排列。返回的必须是格式正确的JSON，不包含任何额外的解释文本。
```

## 4. 分段处理提示词 (SEGMENT_PROMPT)

```
请简要描述这段视频片段的内容，包括场景、人物和文字。请注意这只是完整视频的一部分。必须以JSON格式返回，包含'场景描述'、'人物描述'和'文字内容'三个字段。
```
